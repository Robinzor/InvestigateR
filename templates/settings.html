Processing Options
When enabled, domain names will be automatically resolved to their IP addresses and both will be processed. When disabled, only the domain will be processed.
When enabled, URLs will be automatically resolved to their domain names and both will be processed. When disabled, only the URL will be processed.
When enabled, if a URL host is an IP (e.g., http://1.2.3.4/path), that IP will be processed as an IP observable.
When enabled, result cards with status "Safe", "Not Found", or "Not in Database" will be hidden from the results view.
